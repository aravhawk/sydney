# - PyAudio requires portaudio: brew install portaudio

# - AppleScript used for Mail app integration (built-in)
#
# Linux:
# - PyAudio requires: sudo apt-get install portaudio19-dev python3-pyaudio
# - TTS requires: sudo apt-get install espeak espeak-data
# - Mail integration requires adaptation
#
# Windows:
# - PyAudio binary wheels available via pip
# - TTS requires Windows Speech API or alternative
# - Mail integration requires adaptation
